Tarea:

  1. ¿Qué es un Protocolos Http?

    Es la base de cualquier intercambio de datos en la Web

  2. ¿Qué es un Query String?

    es un término informático que se utiliza para hacer referencia a una interacción con una base de datos

  3. ¿Qué es un Verbos HTTP?

    Son acciones queremos realizar sobre el servidor y son GET, POST, PUT, PATCH, DELETE, HEAD, CONNECT, OPTIONS y TRACE.

  4. ¿Qué es un DNS?

    El DNS es un protocolo de Internet que pertenece a la familia de protocolos de Internet y que se encarga de resolver los nombres de domini

  5. ¿Qué es un CDN?

    Una CDN es una red de servidores que distribuye contenido de un servidor
